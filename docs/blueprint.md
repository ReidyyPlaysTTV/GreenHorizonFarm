# **App Name**: DOC Roster

## Core Features:

- Secure Login: Secure login page for authorized personnel.
- Departmental Dashboard: Centralized home page providing a dashboard view of departmental statuses.
- Interactive Roster: Interactive roster displaying personnel across six departments: Commissioners Office, High Command, Command, NCOS, Corrections, Training.
- Fired/Resigned Listings: Designated page listing personnel who have been fired or resigned.
- DOC Blacklist Management: DOC Command Page featuring blacklists.
- Application Center: Application review center for managing and reviewing applications submitted prior to login.
- Application Form Access: Application form accessible from the login page without requiring login.

## Style Guidelines:

- Primary color: Dark Blue (#003366) provides a sense of authority and security.
- Background color: Black (#000000) for a clean, professional backdrop.
- Accent color: Light Gray (#D3D3D3) for readability.
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a modern, neutral feel.
- Use simple, high-contrast icons for department navigation and personnel actions.
- A clean, well-spaced layout that uses clear divisions and simple information design to promote legibility and ease of navigation.
- Subtle transitions and animations for interactive elements to improve user experience.
